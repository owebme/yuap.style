Buttons

Button sizes

Define a button size with .btn-xs, .btn-s, .btn-m, .btn-l or .btn-xl. The default button size is .btn-m.

Fill button

Fill buttons should be used to signify necessary or mandatory actions. There are two color options, specified with an additional class .btn-primary, .btn-success, .btn-danger or .btn-warning.

{ fillButton() }

Outline button

Outline buttons should be used to signify optional, infrequent or subtle actions. The color options are specified with an additional class .btn-primary-hover, .btn-success-hover, .btn-danger-hover, etc.

{ outlineButton() }

Default button

Default buttons receive an .btn-default class.

{ defaultButton() }

Default button hover

Default buttons should be used to signify optional, infrequent or subtle actions. The color options are specified with an additional class .btn-default-hover-primary, .btn-default-hover-success, .btn-default-hover-danger, etc.

{ defaultButtonHover() }

Uppercase button

Uppercase buttons receive an .btn-upper class.

{ uppercaseButton() }

White button on dark background

This is the reversed version of ghost buttons which is meant to be used on dark backgrounds or reversed themes. The color options are specified with an additional class .btn-white-hover.

{ whiteButton() }

Disabled button

Disabled buttons receive an .btn-disabled class and a disabled attribute.

{ disabledButton() }

Waiting button

Waiting buttons receive an .btn-loading class.

{ waitingButton() }

Button with icon

Buttons with icons are wrapped in a parent component.uiIcon which has to have the button size applied to with an additional class. For example,.button--m will have a parent element with a class .uiIcon--m. The icon has to be placed straight after the button and should be an inline SVG.

{ buttonWithIcon() }

Button with one icon

Unlike the button with icon, the button with one icon doesn't have a parent componet but instead has the.uiIcon class added alongside.uiIcon--m and.button--m classes.

{ buttonWithOneIcon() }

Button group

Button groups consist of an inline set of ghost buttons. Useful for action bars.

{ buttonGroup() }
