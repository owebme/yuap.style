Grid

Page wrap

Page wrap is a wrapper for the whole page. There are 3 different widths, defined by an additional class .pageWrap-s, .pageWrap-m, .pageWrap-l.

{ pageWrap() }

Basic grid

We use grids exclusively for layouts which require a number of columns of equal width. Grids can have 1-12 columns, specified by a class .grid--1, .grid--2, .grid--3, .grid--4, .grid--5or .grid--6. There are also two special case grids with uneven widths which can be initialized by using classes .grid--6-4 and .grid--7-3.
