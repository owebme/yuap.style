Input with icon

Inputs with icons are wrapped in a parent component .input-group-icon which has to have the input size applied to with an additional class. For example, .input-m will have a parent element with a class .uiIcon-m. The icon has to be placed straight after the input and should be an inline SVG.

{ inputWithIcon() }
