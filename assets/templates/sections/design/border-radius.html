Border Radius

Uniform Border radius

Uniform border radius is applied equally to all corners. There are 3 sizes to choose from, .borderRadius-s,.borderRadius-m or.borderRadius-l. Use.borderRadius-circle to make an element circular. All classes have responsive options.

{ uniformBorderRadius() }

Unequal Border radius

Unequal border radius affects only a single corner of the element. The other corners remain unaffected. Again, there are 3 sizes to choose from and all classes have responsive options.

{ unequalBorderRadius() }
