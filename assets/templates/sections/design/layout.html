Uniform spacing

Spacing can be applied using either margin or padding. Both margin and padding share the same predefined scale. There are 6 available spacing sizes; .m-xxs, .m-xs, .m-s, .m-m, .m-l or .m-xl. All classes have responsive options available.

{ uniformSpacing() }

Individual spacing

Individual spacing can be applied to a single side of an element using either margin or padding. Again, both margin and padding share the same predefined scale. The same 6 sizes are available.

{ individualSpacing() }
